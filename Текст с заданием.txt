1. Создать React-приложение на Vite.

Ход выполнения:
- С использованием npm:
    1. npm create vite@latest
    2. Следуйте интерактивным подсказкам:
       Project name: Введите имя вашего проекта (например, my-react-app) и нажмите Enter.
       Select a framework: Используйте стрелки для выбора React и нажмите Enter.
       Select a variant: Выберите нужный вариант:
       JavaScript: Стандартный React с JavaScript.
       TypeScript: React с TypeScript.
       JavaScript + SWC: React с JavaScript и компилятором SWC (еще быстрее).
       TypeScript + SWC: React с TypeScript и компилятором SWC.
       Выберите подходящий вариант (чаще всего начинают с JavaScript или TypeScript) и нажмите Enter.
    3. Перейти в папку проекта cd my-react-app
    4. Установить зависимости npm install
    5. Запустить сервер разработки npm run dev
    6. Открыть приложение в браузере по адресу http://localhost:5173 (или какой был указан в терминале)


2. Разделите макет на компоненты. Например, выделите компоненты страниц:

   Главная
   Избранное
   Страницы фильмов
   Форма добавления фильма
   А также другие более мелкие компоненты.

Создайте компоненты с правильными названиями и текстовыми описаниями компонента
внутри <div> в вашем React-приложении.
